# LED Control using PIC18F4580

## Description
This project demonstrates LED control using a push button with PIC18F4580 microcontroller.

## Components Used
- PIC18F4580
- LED
- Push Button
- Resistor

## Working
- LED is connected to PORTB (RB0)
- Button is connected to RB1
- When button is pressed, LED toggles ON/OFF
- Debounce logic is used to avoid multiple triggers

## Concepts Used
- GPIO (Input/Output)
- Embedded C
- Debouncing
- Polling

## Tools Used
- MPLAB X IDE
- XC8 Compiler
